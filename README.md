# Trabalho-Curriculo

Este projeto é um currículo estilizado e temático inspirado no personagem Thanos, 
desenvolvido como um exercício de HTML, CSS e lógica de organização visual.

O objetivo é apresentar um layout moderno, estruturado e visualmente atraente, 
com elementos personalizados como:

- Fonte temática combinando com o universo Marvel
- Efeito de estrelas animadas usando apenas CSS
- Destaque individual para cada "joia" com cores e efeitos próprios
- Estrutura limpa e semântica, dividida em seções como:
  - Hero (cabeçalho com imagem e apresentação)
  - Sumário (Resumo do personagem)
  - Informações (Hard Skills, Soft Skills, Experiências, Formação, Objetivo e Contato)
  - Rodapé estilizado

A proposta combina design, criatividade e construção de interface, servindo como 
um exemplo de aplicação de estilos avançados em CSS com efeitos visuais dinâmicos,
tudo sem o uso de JavaScript.
